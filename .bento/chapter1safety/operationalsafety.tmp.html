The machine can result in the following dangers if it is used inappropriately or for purposes other than those intended, or else if it is not safe to operate:

Dangers to the safety of the personnel.
Damage to the machine and other material assets of the user.

5.1 Intended use

Machine

The user may use the machine only in the industrial sector.

Installation, operating and transport conditions defined by JFY must be adhered to and maintenance work must be carried out in accordance with the Operator’s manual. The user must observe the regulations of the country in which the machine is being operated as well as national and regional safety and accident prevention regulations.

The user may use the machine to process the materials described in the data collection with the laser. The user may process other materials only after they have been released by JFY.

Standard values for the flatness of metallic materials are EN ISO 9444-2 and EN 10051 for hot-rolled sheets and EN 10131 for cold-rolled sheets.

The following is not permitted:

Any working procedure that impairs the safety.
Processing of combustible materials e.g, plastics, timber, sheet with PVC film, magnesium and others.
Processing of titanium.
The operation of the machine with a laser device that was not delivered with the machine.

Disclaimer

Any use going beyond this is considered to be unauthorized use. JFY is not liable for any damage, especially damage to property, and personal injury as well as production failures resulting from this. The risk is borne solely by the user. The warranty will be voided.

If the operating company modifies or converts the machine, the operating company becomes solely responsible for the safety of the machine and not JFY.

5.2 Significant change to the machine

The machine will be supplied as a complete machine. After commissioning, the responsibility for the safety of the machine is transferred from the manufacturer to the user. The user can produce goods with the machine as intended.

Significant changes

The machine has been significantly changed if a new hazard is posed by this change, which results in an increase in an existing risk or to a new risk which cannot be eliminated or reduced with simple guards.

Simple guards in the context include guards which are integrated in the safety circuit of the machine via signal exchange, or which shut down or shield dangerous machine functions.

The machine can be optionally equipped with signal and safety interfaces ex works, which make it possible for the user or an integrator to add components to the machine, e.g. a robot or fixture or to integrate the machine in a superordinate system.

The user or integrator is responsible for the safety of the altered machine:

Integration may only be performed by qualified personnel
Connection of components to the interfaces must be performed according to the specification set out by JFY.
Personnel must not be endangered by components integrated in the machine.
National regulations must be observed.

Evaluation process

If the user or integrator has made a significant change to the machine, they become the manufacturer of the altered machine.

The user or integrator must carry out an evaluation process(e.g.,conformity assessment) for the altered machine in accordance with national regulations(e.g.,the EU Machinery Directive) of the country in which the machine is operated. He, as the manufacturer, is responsible for the safety of the significantly changed machine.

5.3 Authorized personnel

■ Only authorized and trained personnel may be used for setup and maintenance work.

■ Trained specialist may do the following:

Transporting the machine and its components to the installation site.
Carry out work at laser-specific, hydraulic, pneumatic, and electrical modules.
Disassemble the machine and its components.
