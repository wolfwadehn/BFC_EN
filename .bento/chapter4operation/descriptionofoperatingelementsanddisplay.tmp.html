1.2 Control elements

Control elements on operating panel

Table 1. Table 4- 1
Item No. Operating element Description

1

EMERGENCY STOP button:

In case of a machine emergency, push the EMERGENCY STOP button to switch off the machine drive, control switch and power supply. At the same time, the laser beam is disabled. The machine stays in a FEED HOLD state.

2

Key switch: For connection and disconnection of system power supply.

3

Safety Mode reset:

Push this button to reset or delete alarms. In case of emergency stop of the machine, KA100 safety relay sends an alarm signal. Release the EMERGENCY STOP button. Push reset button to reset.

USB interface：For USB communication.

4

Door Open (safety door lock/unlock button): Used to acknowledge that the door is closed. The indicator is illuminated when the door is closed. Before opening the door, press this button to unlock the door.

Lighting On/Off button: Turn on/off the lighting. The indicator LED is illuminated when the lighting is on.

Dust extractor On/Off button: Turn on/off the dust extractor. The indicator LED is illuminated when the dust extractor is on.

5

Start and stop program for automatic processing. When the program is finished, the machine is stopped automatically.

6

Press the button switch on the laser beam. Release the button to switch off laser beam.

7

Red positioning diode.

8

Read the shutter signal of the laser device to enable and disable laser beam.

9

Output laser beam for co-axial alignment.

10

Press Air/O2/N2 Blowing button, the button turns yellow. Press again to turn off gas blow function.

11

Air indicator is illuminated: assist gas-Air；

O2 indicator is illuminated: assist gas-O2；

N2 indicator is illuminated: assist gas-N2；

12

Activate Z-axis follow function. Used to acknowledge the follow height.

13

Start calibration of the servo.

14

Activate real-time monitoring of machining and loading/unloading to ensure processing safety.

15

Auxiliary function for laser beam settings (e.g. focus position) and other parameter settings e.g. power, pulsing power, peak power.

16

Same as “Return origin” and “Zero point” button on the toolbar. Used for correction of machine origin.

17

“Zero point” button of the cutting head focus adjustment motor for referencing.

18

Switch the Manual/Automatic mode of the pallet changer.

19

Start pallet changer.

Operating elements on pallet changer safety post

Table 2. Table 4- 2
Item No. Operating elements Description

1

In case of a machine emergency, push the EMERGENCY STOP button to switch off the machine drive, control switch and power supply. At the same time, the laser device and hydraulic unit are turned off. The machine stays in the FEED HOLD state.

2

Switch the Manual/Automatic mode of the pallet changer.

3

In Automatic mode of the pallet changer, perform auto exchange of the pallets.

4

In Manual mode of the pallet changer, move the pallet horizontally in a slow speed.

5

”Enable pallet changer” request button. When the indicator LED is illuminated, the pallet changer moves. When the pallets are in positions, the indicator LED is off.

6

Reset safety light barrier.

Note: Blue indicator LED is illuminated when the light barrier is interrupted and the safety relay is off. Press the button to reset safety relay. The blue indicator LED is off.
