Installed operating system

In case of an operating system update of the machine IPC, permission from JFY sales department must be obtained before the update.

Industrial PC

The IPC of the machine can only be used for the operation of the machine. It is not permitted to be used for any other purpose such as office business, e-mail and Internet surfing etc.

Hardware interfaces (e.g. USB interface) must be used in accordance with JFY guide. Connection of any other third-party components to these interface (e.g. connection for charging mobile phone) is strictly prohibited.
