Customer

All conditions described in this chapter must be fulfilled before the machine is delivered.

During start-up, the main switch on the machine may only be turned on by Technical Service.

Technical Service

The commissioning is started up by JFY Technical Service.

The start-up includes the following:

Installing, aligning, leveling.
Laying laser light cables.
Cleaning, rinsing and then, filling cooling circuits.
If necessary, add hydraulic oil.
Connect the machine to the customer-provided media supply (gas, water, compressed air, ...). Only the qualified electrician contracted by the user may establish the power supply.
Performing a functional test.
Instructing personnel.
