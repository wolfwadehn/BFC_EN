Operation process

4.1 Manual mode

Some of the functions of the “Manual control” console are shown as below:

Figure 1. Figure 4- 8

Press the “” button to preview the graphics on the screen. “Up”, “Down”, “Left”, “Right” keys are used to move the axes. On the right are the “Follow Up” and “Follow Down” buttons. In jog mode, press the “’ to switch on the laser beam. Press the “” button to set laser parameters of jog mode.

CypCutE offers software limits function. In “Global parameters”, the function can be activated by selecting the “Enable software limits”option. This function is active by default.

When the range of motion is detected by the system to be exceeding the preset limits, a message will be displayed to warn the user to stop moving the machine to avoid collision. Check the drawing board and make sure that the parts are inside the white boarder.

The current position of the machine is monitored in real-time, the motor will be disabled if the machine is detected exceeding the software limits.

Note: Software limit protection is dependant on correct coordinates. For this reason, each time the machine configuration is changed or the the machine is shut down due to abnormal conditions, the user must execute “Return origin” to calibrate the coordinates.

4.2 Return origin

Click “” button in the CNC menu to move the coordinate axes back to the machine origin and re-establish the coordinate system. See section “Coordinate system” for details. Select “X Only” or “Y Only” to move the axis back to the origin individually. If “Z axis first” or “Auto-focus included” is selected, the corresponding referencing movement will be performed.

4.3 Coordinate system

Modeling coordinate for designing and creating graphic is independent of the machine. The zero point is marked by “”. The coordinate system referenced in machining process and machine positions are shown as below:

Figure 2. Figure 4- 9

Click “Preview” on the console, the cutting head position in relation to the graphic will be displayed.

Mechanical coordinate system

Mechanical coordinate system of the machine is defined by mechanism. Coordinate system is consistent after return origin “” is executed every time. Misalignment after installation or caused by abnormal conditions can also be reset by clicking this button.

CypCutE defines coordinate system with the same rules no matter what mechanism is applied. If you stand in front of the cutting head, towards the right of the cutting head is X+ direction, towards the back of the laser head is Y+ direction, which means the the bottom left position is the minimal coordinate.

Programm coordinate system

Mechanical coordinate is consistent and CypCutE also adopts workpiece coordinate. Workpiece coordinate runs parallel with machine coordinate on X/Y direction but uses a different zero point called “Program 0 point”. Program coordinate systems are divided into floating coordinate system and workpiece coordinate system. Click “” button above the console to select “Floating coordinate”, “Workpiece coordinate” or “External coordinate”. Floating coordinate is used in cutting samples or small quantity parts. Laser head will start processing from current position. The current position of the cutting head is automatically set as 0 point when the user executes “Frame”, “Dry run” or “Start”.

If workpiece coordinate 1-9 is selected, the zero point is manually set by the user, and will only be changed with manual reset. Workpiece coordinate is suited for large quantity production. This will ensure the processing starts from the same position on the machine every time.

Click the status bar “” at the bottom to select mechanical coordinate or program coordinate and specify the zero point of the coordinate. Select “Coordinate positioning” to move the cutting head to the specified coordinate position. For detailed information about determining zero point after interruption, see “CypCutE Laser Cutting Control Software User Manual V1.0”, section 4.1.3.

Zero point should be re-determined after interruption.

4.4 Quick start of cutting

Importing files

Click “ “ button at the left top interface to open the file. Preview window of open file dialog box will help you quickly locate the file you need.

You may also draw graphics with CypCutE software. Click “ ” button, and then use functions on the left toolbar. See the relevant section for details.

  1. AI/DXF/LXD etc. drawing formats are supported.

b.Use “Import” in the menu to import files to the drawing board without deleting the exiting drawings in drawing board.

To open an external file, click “Open”.

Pre-processing

While importing graphics, CypCutE automatically removes minimal graphics and overlapping lines, merges unclosed lines, smooth the curves and sort the groups. Generally, you can start to set cutting parameters without other operations. If the automatic processing cannot meet your requirements, you can open the menu "File" - "User setting" to set the function options you need.

Generally, the software considers that all the graphics to be processed should be closed. If you open a file containing non-closed graphics, a note message will appear in red color. But this function may be turned off. To view the unclosed graphics on the drawing board, you can click “” and “” under “View” button in “Home” menu to highlight the unclosed graphics; you can also select the unclosed grahics by clicking “Select” and “Select Open Figure”.

If you need to split graphics manually, click “Break Line” under “Optimize” drop-down menu and click at the graphic position where you want to split. To merge graphics, select the graphics you want to merge and then click “Join Line” under “Optimize” drop-down menu.

Cutting technique

You may use most of the functions under "Technical Setting" column in the Home menu bar, including lead line setting and compensation, etc. The large button“Lead”can be used to set lead lines. Button“Seal”is used to set over-cut, gap, seal or multi-cut parameters. Button “Compensation” is used for kerf compensation. Button “Micro Joint” is used to inset uncut micro-joints in graphics. Button “Reverse” is used to reverse the machining direction of a single graphic selected. Button “Cooling Point” is used to set cooling point. Click “Start Point” button and point at any position on the graphic where you want the cutting to start to set it as a start point. If you point at a position outside the graphic, then a lead line is set from the the position to the graphic.

Press Ctrl+A to select all graphics. Click "Lead" to set parameters and the program will set lead line on graphics automatically. Click "Lead" to open the drop-down menu and select "Check Leadline". The program will check for lead lines intersecting with the graphic or in wrong direction and optimize them.

On the right toolbar, click “Layer” button to set detailed cutting parameters. The “Layer Parameter Settings” dialog box include basically all the parameters related to cutting performance.

Cutting path planning

Set the cutting sequence of graphics as needed. Click “Sort” in “Home” or “Nest” column to set the cutting sequence of all graphics automatically. Open the drop-down menu and choose the sorting pattern to decide if processing direction is allowed to be changed by automatic sorting and identify the inner and outer contour of the part graphic.

If automatic sorting does not meet the requirements, click “123” button to enter the manual sorting mode.Click the graphic with the mouse in sequence to set the processing order. Or press and hold the mouse to draw a line from one graphic to another to specify the order between the two diagrams.

Select several graphics that have been sorted (set sequence) and click “Group” button in “Home” or “Nest” column to fix their order. Afterwards, the sequence within the group will not be changed with either automatic or manual sorting. The group will always handled as a single complete unit.

Right click a group and select “Group Sort” to change the sequence of the graphics within the group.

Pre-cutting check

Check the processing path before cutting. Drag the progress bar to quickly view the graphic processing sequence. Click “Interactive preview button” so you can view the graphic processing order one by one.

Figure 3. Figure 4- 10

Click “Simulate” button on the console to simulate processing. The “Simulate Speed” function on “CNC” page allows you to adjust the simulation speed.

Figure 4. Figure 4- 11

Start cutting

Please note that a compatible control card and dongle are required to perform the actual processing.

Before start processing, you have to check the relative position of the graphic to the working area. Click “” button on the left of the console to view the graphic position in relation to the working area. This relation is calculated by the position of the “Ref point” on the screen related to the laser head position at machine. On the screen are some common position marks used in our coordinate system. Click “Preview” and the “Ref point” will move the laser head position.

If the laser head position indicated by the red cross does not match with the actual position of the laser head, please check if the machine origin is correct or not. If not, the machine origin can be corrected by selecting “CNC” and “Return”. If the graphic is partially or completely outside the white frame in Preview mode, it indicates that the travel range of the machine may be exceeded.

Click “Ref point” at Home menu to change the “Ref point” position. For example, if the laser head is at the left bottom position of the graphic, set the the left bottom position as “Ref point” position.

Figure 5. Figure 4- 12

After checking the position, click “” button on the console, the laser head will move along the outer rectangular frame of the graphic to be processed. This can be used to check if the machining area on the machine is right. Click “Generate Envelope” on the menu bar to generate the envelope. Then click “” button to check the positions of the remnant parts with irregular contours. Click “” button to move the cutting head along the entire cutting path with laser off and check for cutting issues that may occur in a more detailed way.，

Execute the program. Click “” button to start processing. During the processing, a monitoring panel will be displayed as shown in the figure below, including parameters such as speed, power, follow height, focus, processing timing and processing file name etc. Click “” button to pause the processing. In pause status, you can manually move the laser head up and down, manually switch on/off the beam, gas etc. Click “” to resume processing.

Click “” button to stop processing. The laser head will return to the pre-set position.
