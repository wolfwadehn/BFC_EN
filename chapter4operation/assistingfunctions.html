Assisting functions

Assisting functions

5.1 Simulation

Users can simulate the cutting process to check the cutting sequence in program. This process is executed in software only and no command will be sent to the machine. By simulating the processing, users can check the sequence between parts and in parts.

Click “Simu” in console. It will be automatically switched to“CNC”menu. You can adjust simulation speed in tool bar.

5.2 FindEdge

CypCutE supports two patterns of sheet edge locating: BCS100 Find Edge and manual edge location. Click “Find Edge” and in the drop-down menu you can select the find-edge pattern to locate sheet position and edge. Find edge result will be displayed on the top right area of the drawing board.

By clicking “Sizing” button under “CNC” tab or clicking “Capacitance Sizing”, you can open the capacitance sizing interface.

Sheet size X refers to the sheet length on X direction; sheet size Y refer to the sheet length on Y direction. The 2nd and 3rd positions to locate the sheet edge is calculated with the sheet dimension. It’s recommended to input values slightly lower than actual dimensions to avoid laser head collision with the slat teeth.

Make sure that the parameters values are matched with the measured sheet dimensions. It’s recommended to set the values slightly smaller than the measured dimensions. Wrong setting may lead to cutting head collision with the slats.

Margin: Find-edge result will determine a 0 point of the sheet. This margin is used to adjust the 0 point inward or outward the sheet. If you have set margin in nesting plate, you can input 0 here.

Identify sheet dimensions: set sheet dimensions automatically according the border of the graphic

6-point Find Edge: Recommended for use on thin sheet. Laser head will execute 6 times find-edge action to reduce the interference of the slat teeth.

Start from preset position: You can set a fixed position from which the find edge process start every time. You can define a proper position as the fixed position for the laser head. This position must be within the sheet.

Start from current position: Laser head will start finding edge from current position.Please make sure that the laser head is within the sheet.

If “find edge before machining” is selected, the machine will execute FindEdge before machining.

Note: Before executing the Find Edge function, please make sure that the laser head is able to follow the material properly. The inclination angle of the sheet should not exceed 10 degrees.

Find edge parameter window is shown as below:

Figure 2. Figure 4- 15

You can also set more detailed actions by clicking “Detail settings” at bottom left corner.

Figure 3. Figure 4- 16

Find edge speed affects the accuracy of result, recommended 100mm/s.

Find edge height is the distance between the nozzle and sheet during the find-edge process.

Prong Dif is the gap between the adjacent tooth tips of the pallet slat.

2-point distance and 2-Pt (short side) decide positions of the other two points.

5.3 Frame

Click the “’” button on the console. The laser head will move along the outline of graphic to be processed in a rectangular frame.This will help the users identify the machining position and workpiece dimensions. To set the frame speed, go to ”Layer”, “Global parameters” and “Frame speed”.

Note: If find edge is executed before frame operation, the find-edge result will be saved. When executing frame, the laser head will move along a tilted rectangle which is corrected by the find-edge result. See “Find Edge” section for details.

5.4 Start and Dry Run

Click “” to start machining. The machining process will be monitored for parameters such as coordinates, speed, time count and follow height etc.

Figure 4. Figure 4- 17

When the above status is displayed, you can not switch to other pages except the File menu to secure the graphics against being modified by mistake when machining. You can pause the machining to change parameters, which will take effect once the machining is resumed. If the change is made without pausing the machining, the new settings will take effect from the next machining.

Click “” button, the laser head will move along the actual cutting path with the same speed and acceleration. But the laser, gas and follow function are turned off. Meanwhile, other functions such as pause, resume, forward, backward and break point information etc remain the same as the actual cutting. Dry run is used to check the machining process without cutting the sheet.

By default setting, laser head will return to zero point after finishing machining. If you want the laser head to return to a specified position, you can choose from zero point, start point, end point, origin and mark points om the control panel.

Figure 5. Figure 4- 18

If “” is not selected, laser head will stand still at the finish position of the machining. If you want the laser head to return to a mark point, select “Process ends, go to” + desired position.

After a machining process ends, 1 will be added up in “Finished” on the console panel. When “Finished” value reaches “Total” value, a prompt message will appear to remind the users. When the machining process is completed, you can set target number of machining and laser head actions in “”. For cycle processing, click “” button to complete the setting.

5.5 Stop, Pause and Resume

To stop processing, click “” or “” button on the console.

Click “” or “” button on the console to pause the machining process. Meanwhile, you can modify parameters in layer window, or execute the functions on the console panel such as burst laser, turn on/off gas and follow function etc.

Click “” or “” button on the console to resume the machining process from interrupted position. Click “” button will resume the machining process without piercing.

Click “” or “” button to move the laser head backward or forward along the cutting path.The step size and speed can be set at “” on the console.

5.6 Break point

When the machining process is interrupted by accident or stops, the break point will be saved by software. As long as the graphics and parameters are not modified, you can click the “” button to locate the break point automatically. Click the “” button and the program will resume machining from the break point.

If related parameters are changed after the machining process is stopped, the “Start” button will be displayed with an asterisk sign “ ”. When “” appears, break point functions will not be available.

5.7 Start at a specified position

CypCutE allows the users to start machining from any specified position. Right click on the graphic where you want to start machining and select “Start from here” as shown below:

Figure 6. Figure 4- 19

For safety concern, after “Start from here” is selected, a confirmation dialog box will pop up again. After confirmation, the laser head will move to the specified position and then start machining. The path before the specified position will not be processed.

If you want the laser head to stop at the specified position only and not to start machining, select “Locate here”. The laser head will move to the specified position and enter the pause mode.

You can execute”Locate here” multiple times with right click until the desired position is reached. Or you can click “” and “” to locate a position more precisely.

5.8 Global parameters

Under “Global parameter” tab in the “Layer parameter adjustment” dialog box, you will find some motion control parameters which can be modified. Modification of these parameter settings will affect the machine running condition, cutting performance and efficiency.

Below are part of the parameters listed in “Global parameters” tab:

Table 1. Table 4- 4
Motion Control Parameters

Travel speed

Dry run speed (different from cutting speed)

Travel acceleration

Max axis acceleration of on dry run. Used in combination with travel speed.

Travel low-pass frequency

Travel acceleration jerk on dry run. Used in combination with travel acceleration.

Frame speed

Speed of frame-walk on outline of the workpiece.

Cutting acceleration

Max axis acceleration in cutting; Used in combination with cutting speed.

Cutting low-pass frequency

Acceleration jerk in cutting. Used in combination with cutting acceleration.

Jog acceleration

Acceleration in jogging.

Jog low-pass frequency

Acceleration jerk in jogging. Used in combination with jog acceleration

Corner acceleration

The acceleration at corner position when travel direction changes.

Curve precision

The smaller the value, the higher the machining accuracy of the curve.

Corner precision

The smaller the value, the higher the machining accuracy of corners.

Default Settings

Gas-on delay

Blowing delay before cutting.

First gas-on delay

Delay when open gas first time after machine start-up.

Pre gas on

Pre-blowing time when switching the gas.

Gas switch delay

he delay time required to make sure the old gas is totally discharged and the new gas fully fill the gas circuit

Cooling delay

The time of blowing gas at cooling point.

System delay

Delay of servo response

Resume, stepback

The distance traveled by the laser head back to the cutting path when resuming the machining process from pause status

Gas Parameters

Gas pressure

Gas pressure when performing blowing.

Gas on

Duration of a single blowing

Interval

Gas shut-off of gas flushing process.

Start, Gas on

Blowing times at the beginning of processing,

Resume, Gas on

Blowing time when resuming processing.

Follow Control Parameter

Lift height

Maximum height of frog leap

Max follow height

There is an upper limit of the follow height. When this max follow height is required, e.g. for piercing, Z axis will move in 2 steps: Z axis will follow to a close position to the sheet surface and then lift up to the max follow height. This parameter is used to set the upper limit of the follow height.

Frog leap lift

Travel on X/Y direction without Z axis lifted to the correct position in order to save time

Smart Frog Leap

Automatically adjust the frog leap height according to the tilting conditions of the workpieces.

Frog-leap start height

Lift to the set height before execute frog leap

Max frog jump distance

The distance from the start of the frog jump.

Short distance without lifting

When the length of air movement is less than this length, the elevator does not lift up during air movement, and keeps following state.

Hold fixed height

Suitable for non-metal cutting. If this option is selected, follow function will be disabled and only location function available

Smart follow

Short-distance travel without sheet detection

Preset Parameters

Enabling NURBS spline interpolation

After checking it, NURBS fitting will be performed adaptively for the machining curve, which can improve the machining speed and smoothness of graphics.

FlyCut at MicroJoint

No pause at microjoints wutg fly cutting. Invalid when piercing techniques are enabled.

Technique check

Check whether the default process parameters are valid.

Thickness check

Check whether the material thickness information is set in the process parameter interface.

Quick Find Edge

Reduce the number of the safe edge-searching processes.

Edit PLC Process

PLC process editing.

Pierce Defilm

Set the cutting sequence for deburring and pre-piercing.

FlyCut overcut

Overfly cut distance which ensures that part is cut smoothly.

Extended cufoff length

The length of over-cut when cutting cut-off line.

Gas on protection

Select “Blow” in layer settings, and set gas pressure here.

Unit

Selection of the unit of parameters according to your preference.
